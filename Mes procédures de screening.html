Votre navigateur n’est plus à jour ?

Vous visitez notre site web avec une ancienne version de Microsoft Internet Explorer (version 8 ou antérieure). Afin de consulter travaillerpour.be (et d’autres sites web) le plus efficacement possible, nous vous conseillons d’installer une version plus récente. L’installation est gratuite. Installez la version la plus récente de Microsoft Internet Explorer (version 11). Vous aurez ainsi plus facile à visiter notre site (et d’autres sites).

Téléchargez la version la plus récente d’Internet Explorer. Le navigateur s’ouvre dans une nouvelle fenêtre. Cliquez sur « Téléchargez Internet Explorer 11 » et suivez les instructions sur le site de Microsoft. Téléchargez la version la plus récente ici : Microsoft.
